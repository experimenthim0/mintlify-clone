# Mintlify Landing Page Clone

A high-fidelity, modern landing page clone of **Mintlify** — the intelligent knowledge platform. This project focuses on capturing the "AI-native" aesthetic of the 2026 web, featuring dark mode layouts, clean typography, and responsive design.

##  Demo
https://mintlify-clone-nine.vercel.app/

##  Features
* **AI-Native UI:** Dark-themed interface optimized for developer tools and SaaS products.
* **Feature-Rich Sections:**
    * **Hero Area:** Animated-style "New" badge and email capture.
    * **Brand Wall:** Responsive logo section featuring Perplexity, X, and more.
    * **Intelligent Features:** Dedicated blocks for LLMs.txt, MCP, and Agentic workflows.
    * **Enterprise Grade:** Security-focused layout highlighting SOC 2 and compliance.
* **Modern Footer:** Comprehensive multi-column footer with social icons and system status monitoring.
* **Fully Responsive:** Built to adapt across various screen sizes using modern CSS techniques.

## Tech Stack
* **HTML5:** Semantic markup for structure.
* **CSS3:** Custom styles using Flexbox and Grid.
* **Font Awesome 7.0:** Scalable vector icons for a professional look.
* **BrandFetch Website :** High-quality logo assets.
